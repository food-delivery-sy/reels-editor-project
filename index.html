محرر قوالب الفيديو المتحركة

أداة متقدمة لإنشاء وتحرير قوالب الفيديو المتحركة بواجهة سهلة الاستخدام

نبذة عن المشروع

محرر قوالب الفيديو المتحركة هو تطبيق ويب متقدم مبني باستخدام React وTypeScript، يتيح للمستخدمين إنشاء وتحرير قوالب فيديو متحركة بسهولة. يدعم التطبيق اللغة العربية واتجاه RTL بشكل كامل، ويوفر واجهة سهلة الاستخدام مع العديد من الميزات المتقدمة.

الميزات الرئيسية

تحرير الشرائح

إمكانية إضافة وتحرير وترتيب الشرائح بسهولة، مع دعم أنواع مختلفة من المحتوى.

تخصيص المظهر

تعديل الألوان والخطوط والأنماط لتناسب احتياجاتك وهويتك البصرية.

تصدير الفيديو

إمكانية تصدير المشروع كفيديو متحرك بجودة عالية وبتنسيقات مختلفة.

دعم اللغة العربية

واجهة كاملة باللغة العربية مع دعم الاتجاه من اليمين إلى اليسار (RTL).

حفظ وتحميل المشاريع

إمكانية حفظ المشاريع واستعادتها لاحقاً، مع دعم التخزين المحلي.

واجهة سهلة الاستخدام

تصميم بديهي يسهل على المستخدمين من جميع المستويات استخدام التطبيق.

زيارة المستودع على GitHub

التقنيات المستخدمة

  • React.js و TypeScript
  • Vite كأداة بناء
  • Tailwind CSS للتنسيق
  • Shadcn UI لمكونات واجهة المستخدم
  • FFmpeg لمعالجة الفيديو

كيفية الاستخدام

لاستخدام المشروع محلياً، يمكنك اتباع الخطوات التالية:

  1. استنساخ المستودع: git clone https://github.com/food-delivery-sy/reels-editor-project.git
  2. تثبيت التبعيات: npm install
  3. تشغيل المشروع: npm run dev
